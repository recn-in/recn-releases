# RECN Rec — Releases

Installers for **RECN Rec Desktop**. RECN is currently distributed as beta
software; there is no stable channel yet.

- **macOS Beta:** [RECN-Rec-Beta-macOS.dmg](https://github.com/recn-in/recn-releases/releases/download/v0.0.1-beta/RECN-Rec-Beta-macOS.dmg) — signed + notarized, drag to Applications.
- **Windows Beta:** [RECN-Rec-Beta-Windows-Setup.exe](https://github.com/recn-in/recn-releases/releases/download/v0.0.1-beta/RECN-Rec-Beta-Windows-Setup.exe) — installs per-user, no admin required.

Beta Nightly builds contain the newest merged changes and are available from
the [releases page](https://github.com/recn-in/recn-releases/releases).

Website: [recn.app](https://recn.app)
